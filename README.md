# CryptoCandleDataToCSV

This application fetches historical candlestick data for specified cryptocurrency trading pairs and granularities from the Coinbase API, processes the data to remove duplicates, and exports it to CSV files for further analysis.

## Features

- Fetches historical candlestick data for multiple cryptocurrency pairs.
- Supports various granularities (e.g., 1 day, 1 hour, 15 minutes, 5 minutes).
- Removes duplicate entries from the data.
- Exports processed data to CSV files.


## Dependencies

- Coinbase Advanced Trade API
- CsvHelper
